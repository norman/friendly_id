    Attributes

    cache_column[RW]

    The column used to cache the friendly_id string. If no column is specified, FriendlyId will look for a column named cached_slug and use it automatically if it exists. If for some reason you have a column named cached_slug but don't want FriendlyId to modify it, pass the option +:cache_column => false+ to {FriendlyId::ActiveRecord2#has_friendly_id has_friendly_id}.

    child_scopes[R]

    An array of classes for which the configured class serves as a FriendlyId scope.
